# Node.js Hello World

A simple Node.js Hello World application that demonstrates basic Node.js functionality.

## Features

- Prints "Hello, World!" to the console
- Creates a simple HTTP server that responds with "Hello, World!"
- Runs on port 3000 by default (configurable via PORT environment variable)

## Requirements

- Node.js (version 12 or higher)

## Installation

No additional dependencies required. This application uses only Node.js built-in modules.

## Usage

### Run the application:

```bash
npm start
```

or

```bash
node app.js
```

### Access the web server:

Open your browser and navigate to `http://localhost:3000`

## Scripts

- `npm start` - Start the application
- `npm run dev` - Start the application in development mode

## Output

When you run the application, you will see:
1. "Hello, World!" printed to the console
2. "Server running at http://localhost:3000/" message
3. The web server will be accessible at the specified URL
